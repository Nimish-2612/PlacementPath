# **App Name**: PlacementPath

## Core Features:

- User Authentication: Secure email and password based authentication, with Firebase Authentication.
- Dashboard Summary: Display summary cards for DSA Progress, Projects Completed, Core CS Coverage, and Placement Readiness Score.
- Placement Roadmap: Structured roadmap for DSA, Core CS, and Projects with checkable items.
- DSA Tracker: Topic-wise checklist with status markers (Not started, In progress, Confident) and auto-calculated completion percentage.
- Project Tracker: Add projects with name, tech stack, confidence level (Low, Medium, High), and placement readiness indicator.
- Placement Readiness Score Tool: Generate a score based on DSA completion, Core CS completion, and project confidence. This is a tool that uses an LLM to intelligently combine the three metrics.

## Style Guidelines:

- Primary color: Sienna (#A0522D) for warmth and approachability, inspired by the themes of education and growth.
- Background color: Off-white (#F5F5DC), a desaturated version of sienna in a light color scheme.
- Accent color: Goldenrod (#DAA520) to highlight important information. A hue shifted to the left of sienna, with brighter saturation to create contrast.
- Body: 'PT Sans', a modern, humanist sans-serif. Headlines: 'Playfair', a stylish serif well-suited for headlines, with 'PT Sans' as body.
- Minimalist line icons for categories and actions.
- Clean, modern layout with rounded cards and soft shadows. Light and dark mode toggle.
- Subtle transitions and animations for user interactions.
